# NY-restaurant-review


## Dependencies:

* Font-Awesome
* EJS
* EJS-Mate
* Express
* MySql
* Nodemon

Note: All project dependencies will be installed via the `npm install` command.

## Setup guide:

1. Clone this repo
1. Run `npm install`
1. Run `nodemon app.js` to start the server
1. Navigate to `http://localhost:8080/`
